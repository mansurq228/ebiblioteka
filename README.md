# ebiblioteka
w library
Проект "Эбиблиотека"
Сайт представляет собой электронную библиотеку, в которой пользователь может:
- просматривать список книг;
- фильтровать книги по жанрам;
- добавлять новые книги через форму;
- переключать светлую и тёмную тему;
- переходить между страницами сайта.
Проект выполнен с использованием HTML, CSS и JavaScript без сторонних библиотек.
Структура проекта:
ebiblioteka/
├── index.html // главная страница
├── pages/ // остальные страницы сайта
├── css/ // стили
├── js/ // JavaScript-скрипты
├── images/ // изображения
└── README.md
